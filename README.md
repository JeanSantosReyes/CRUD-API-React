Instalaciones:
* React Router Dom
* Axios
* Bootstrap
* [React Table](https://www.npmjs.com/package/react-table)
* FontAwesome

```
npm i react-router-dom

npm i axios

npm i bootstrap@5.3.0-alpha1

npm i react-table
npm i --save-dev @types/react-table
```
